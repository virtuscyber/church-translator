🎛️ Dante Via Setup Guide

If you're using Dante Via to route audio from your Dante network:

  1. Open Dante Via on this machine
  2. In the Audio Device tab, find the Dante channel carrying the preacher's mic
  3. Click the ▶ button next to it — this routes it to a virtual input device
  4. The virtual device will appear in the Microphone dropdown above
  5. Click 🔍 Scan All Inputs — speak into the mic and look for the green bar
  6. Select that device, save settings, and start translating!

Tip: Dante Via devices typically show up as "Dante Via Receive" or similar. The scan makes it easy to find — just look for the one with the live audio bars.

Without Dante Via: If you're using a USB mic or laptop mic directly, just scan and select the device showing audio.
